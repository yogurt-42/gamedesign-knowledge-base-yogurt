# 非生物威胁系统

> 环境作为主动威胁的设计模式

## 核心设计原则

**非生物因素(Abiotic Factors)** 不仅是背景，而是**主动的游戏机制**。

> "玩家不是在对抗怪物，而是在对抗环境本身。"

---

## 威胁类型库

### 1. 温度崩溃 (Temperature Collapse)

**机制模型**：
```
环境温度 ← 区域基础值 + 天气事件 + 设施状态
    ↓
玩家体温 → 失温/中暑 → 能力 debuff → 死亡
    ↓
科学应对 → 供暖/制冷设备 → 资源消耗
```

**设计参数**：
| 参数 | 说明 | 默认值建议 |
|------|------|-----------|
| 体温范围 | 安全/警告/危险/致命 | 36-37°C / 35-38°C / 32-39°C / <32或>40°C |
| 崩溃速度 | 温度偏离后的恶化速率 | 每分钟 ±0.5°C |
| 恢复速率 | 回归安全区的恢复速度 | 每分钟 +1°C |
| 设备效率 | 供暖/制冷的效果 | 每设备 +2°C/分钟 |

**视觉反馈**：
- 安全：正常视野
- 警告：边缘轻微模糊
- 危险：呼吸白雾/热浪扭曲
- 致命：屏幕结冰/灼伤效果

---

### 2. 供电中断 (Power Failure)

**基础设施依赖模型**：
```
发电机 → 变压器 → 区域配电 → 设备运行
    ↓         ↓          ↓
 燃料       损耗      优先级管理
```

**连锁效应**：
| 断电区域 | 直接后果 | 间接后果 |
|---------|---------|---------|
| 照明系统 | 视野受限 | 理智下降、伏击风险 |
| 收容单元 | 安全协议失效 | 收容物活跃++ |
| 门禁系统 | 无法通过 | 被困/绕路 |
| 分析设备 | 无法研究 | 科技停滞 |

**玩家决策**：
- 分配有限电力到哪些系统？
- 是否冒险启动备用发电机（可能吸引注意）？

---

### 3. 能见度危害 (Visibility Hazards)

**黑雾/烟雾/辐射尘机制**：

**层次化视野**：
```
完全可见 (5m内) → 模糊轮廓 (5-10m) → 完全黑暗 (>10m)
      ↑                  ↑                  ↑
  正常交互         仅能识别大体类型      无法感知
```

**动态变化**：
- 黑雾浓度随时间波动
- 某些收容物会释放/吸收雾气
- 灯光可驱散小范围黑雾（但吸引注意）

---

### 4. 辐射与污染 (Radiation & Contamination)

**累积性伤害模型**：
```
暴露时间 × 辐射强度 = 累积剂量
      ↓
   健康惩罚（线性增长）
      ↓
   突变风险（阈值触发）
      ↓
   慢性/急性辐射病
```

**防护层级**：
1. **时间限制**：快速通过
2. **距离衰减**：平方反比定律
3. **屏蔽材料**：铅板、混凝土、水
4. **净化设施**：事后处理

---

### 5. 不稳定重力场 (Unstable Gravity)

**方向/强度变化**：
- 局部重力反转（天花板变地板）
- 重力减弱（跳跃过高、投掷物轨迹改变）
- 重力增强（移动困难、坠落伤害++）

**涌现性玩法**：
- 利用重力异常到达常规无法到达的区域
- 投掷物在异常区的非预期轨迹
- 收容物行为受重力影响改变

## 威胁叠加设计

### 组合效应表

| 威胁A | 威胁B | 叠加效果 | 应对策略 |
|-------|-------|---------|---------|
| 寒流 | 断电 | 供暖失效，失温加速 | 紧急生火、保温材料 |
| 辐射 | 黑雾 | 无法看到辐射源，双重恐惧 | 辐射探测器+缓慢移动 |
| 重力异常 | 供电中断 | 移动困难+门禁失效 | 固定路线、备用出口 |

**完美风暴**：
- 三重以上威胁同时激活 = 紧急撤离模式
- 需要团队配合或牺牲部分资源

## 科学应对机制

### 检测 → 分析 → 应对 循环

```
1. 检测：仪表/症状显示异常存在
2. 分析：确定威胁类型和参数
3. 方案：选择科学应对措施
4. 执行：消耗资源/时间实施
5. 验证：效果评估，必要时调整
```

### 应对工具示例

| 威胁 | 科学工具 | 原理 |
|------|---------|------|
| 寒流 | 冷聚变反应堆 | 高效产热 |
| 辐射 | 铅屏蔽+碘片 | 阻挡+代谢 |
| 黑雾 | 紫外线灯 | 光分解 |
| 重力异常 | 重力锚定器 | 局部稳定 |
