# 办公空间资源化

> 将日常办公设施转化为生存资源的系统设计

## 核心设计原则

**不是从自然中采集，而是从文明遗迹中回收。**

> "在这个设施里，你不是猎人，你是考古学家——挖掘文明的残骸来维持生存。"

---

## 拆解系统

### 基础拆解逻辑

```
环境物体 → 右键/交互拆解 → 获得原材料 + 信息线索
```

### 拆解示例表

| 源物体 | 产出资源 | 附加信息 |
|--------|---------|---------|
| **办公椅** | 木材(2) + 金属管(1) | 前员工姓名牌（叙事线索） |
| **电脑主机** | 电路板(1) + 金属碎片(2) | 硬盘（可能恢复数据） |
| **办公桌** | 木材(3) + 抽屉把手(1) | 抽屉内文件（剧情碎片） |
| **饮水机** | 塑料(2) + 过滤芯(1) | 最后更换日期（时间线索） |
| **复印机** | 金属(3) + 电路板(1) + 墨粉(1) | 复印记录（谁最后使用） |
| **灭火器** | 金属罐(1) + 压缩气体(1) | 检查标签（维护记录） |
| **荧光灯** | 玻璃(1) + 电子元件(1) | — |
| **文件柜** | 金属(2) | 柜内档案（重要文档） |

### 拆解层级

**Level 1: 表面拆解**
- 快速获得基础资源
- 不破坏物体主体结构
- 示例：拆下椅子腿，但保留座椅框架

**Level 2: 深度拆解**
- 需要工具（螺丝刀、扳手）
- 获得更多资源 + 内部组件
- 示例：完全拆解电脑，获得硬盘、内存条、CPU

**Level 3: 破坏式拆解**
- 使用暴力（砸碎、切割）
- 资源产出++，但可能损坏有价值的内部物品
- 示例：砸碎文件柜，获得金属但可能毁坏档案

## 场景考古学

### 叙事性拆解

拆解不仅是资源获取，更是**信息发现**的过程。

**示例场景**：
```
办公桌抽屉
├── 上层：办公用品（笔、便签）→ 资源
├── 中层：个人物品（照片、零食）→ 叙事线索  
└── 底层：机密文件（标有"仅内部"）→ 剧情推进
```

### 时间层设计

同一区域在不同"时间层"有不同状态：

| 时间层 | 状态 | 可拆解物 | 叙事信息 |
|--------|------|---------|---------|
| **事发前** | 整洁有序 | 标准办公用品 | 日常工作记录 |
| **事发时** | 混乱 | 匆忙遗留物 | 紧急疏散痕迹 |
| **现在** | 破败 | 损坏设备 | 事故后果 |

**动态变化**：
- 玩家首次进入：事发前状态（通过某种方式，如监控录像）
- 返回时：现在状态
- 对比两个状态，推断发生了什么

## 90年代科幻设施美学

### 视觉风格指南

**色彩**：
- 主色调：米色、灰褐色、 institutional green（机构绿）
- 点缀色：CRT显示器的荧光绿/琥珀色
- 警告色：褪色的红色警示标签

**材质**：
- 水磨石地板（磨损）
- 纤维板天花板（发黄）
- 金属门框（生锈）
- 软木板公告栏（褪色的通知）

**标志性物品**：
- CRT显示器（巨大、沉重）
- 软盘盒（标签手写）
- 传真机（卷纸）
- 打孔机、订书机（工业级）
- 咖啡杯（印有机构logo，积累茶渍）

### 时代感细节

**技术痕迹**：
- "千年虫"准备海报
- 拨号上网的提示音（某些老式电脑）
- Windows 95/98 风格的界面（如果有电脑可用）
- 日期戳：1998-1999年

**文化参考**：
- 《X档案》风格的保密氛围
- 《半条命》黑山基地的审美
- 冷战末期/后冷战早期的官僚感

## 平凡物品的异常用途

### 非预期使用

| 原用途 | 物品 | 异常用途 | 科学原理 |
|--------|------|---------|---------|
| 办公 | 订书机 | 发射装置 | 弹簧势能 |
| 清洁 | 吸尘器 | 气动武器 | 气压差 |
| 餐饮 | 微波炉 | 电磁干扰器 | 微波辐射 |
| 安全 | 灭火器 | 喷射武器 | 高压气体 |
| 通讯 | 电话机 | 电击陷阱 | 变压器 |
| 记录 | 碎纸机 | 切割工具 | 旋转刀片 |

### MacGyver 式创新

**设计理念**：
> 玩家应该能够像《百战天龙》MacGyver 一样，用胶带和回形针创造奇迹。

**示例**：
- 胶带 + 金属管 + 打火机 = 简易喷灯
- 电线 + 电池 + 水桶 = 电击陷阱
- 镜子 + 阳光 + 纸片 = 点火装置

## 空间功能重定义

### 房间改造

玩家可以将办公空间重新定义为生存设施：

| 原功能 | 房间 | 可改造为 | 所需资源 |
|--------|------|---------|---------|
| 会议室 | 大空间 | 作战室/指挥中心 | 桌子→地图台、白板→战术板 |
| 茶水间 | 小空间 | 医疗站 | 冰箱→药品冷藏、微波炉→消毒 |
| 档案室 | 封闭 | 安全屋 | 文件柜→路障、防盗门加固 |
| 实验室 | 设备齐全 | 科研中心 | 已有设备，只需供电 |
| 服务器房 | 温控 | 冷藏库 | 改造成食品存储 |

### 基础设施利用

**通风系统**：
- 正常用途：空气流通
- 异常用途：快速移动通道、毒气输送、声音传播

**电力系统**：
- 正常用途：供电
- 异常用途：电击陷阱、电磁脉冲、过载破坏

**管道系统**：
- 正常用途：供水/排水
- 异常用途：秘密通道、液体输送（毒/燃料）

```
┌─────────────────────────────┐
│  [办公椅]                    │
│  ╔══════════════╗          │
│  ║  [椅子图像]   ║          │
│  ╚══════════════╝          │
│                             │
│  拆解选项：                   │
│  [表面拆解] → 木材(1)        │
│  [深度拆解] → 木材(2)+金属(1)│
│  [暴力拆解] → 木材(2)+金属(1)│

│                             │
│  发现物品：                   │
│  [员工名牌] 查看             │

```

### 视觉反馈

**可拆解物体高亮**：
- 鼠标悬停：轻微发光
- 可拆解：绿色轮廓
- 需要工具：黄色轮廓（显示所需工具图标）
- 不可拆解：红色轮廓

**拆解过程动画**：
- 螺丝旋转飞出
- 木板断裂分离
- 金属弯曲变形
