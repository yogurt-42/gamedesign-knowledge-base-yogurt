# CASE-001: Abiotic Factor (非生物因素)

**案例编号**: CASE-001

## 1. 核心设计理念

### 1.1 身份重构：科学家而非战士

**核心突破**：生存游戏的能力模型从"武力"转向"智力"。

| 维度 | 传统生存游戏（战士/探险家） | 非生物因素（科学家） |
|------|----------------------|------------------|
| **核心属性** | 力量、敏捷、耐力 | 智力、感知、知识 |
| **战斗方式** | 武器伤害、近战格斗 | 环境利用、陷阱、规避 |
| **资源获取** | 采集、狩猎、掠夺 | 拆解、分析、合成 |
| **问题解决** | 直接对抗 | 科学方法 |

**关键洞察**：玩家是科研人员，核心能力是**知识应用**而非武力。

> "在这个设施里，一把枪救不了你。能救你的是对物理定律的理解。"

### 1.2 非生物因素：环境作为主动威胁

游戏标题《Abiotic Factor》（非生物因素）直接指向生态学概念——环境中影响生物的非生命要素（温度、光照、辐射等）。

**设计原则**：非生物因素**不仅是背景，而是主动的游戏机制**。

| 威胁类型 | 机制效果 | 科学应对 |
|---------|---------|---------|
| **寒流** | 温度骤降，快速失温 | 供暖设备、保温服、冷聚变堆 |
| **供电中断** | 收容单元失效、门禁锁定 | 备用发电机、手动维护、电力分配决策 |
| **黑雾** | 视野受限，理智下降 | 光源、紫外线灯、快速撤离 |
| **辐射** | 持续伤害，变异风险 | 铅屏蔽、半衰期计算、抗辐射药 |
| **不稳定重力** | 跳跃/移动受影响，投掷物轨迹改变 | 重力锚定设备、利用异常到达新区域 |
| **气压变化** | 门无法开启/突然关闭 | 气压调节、备用通道 |

**威胁叠加设计**：
- 单一威胁：可管理
- 双重威胁：需要优先级判断
- 三重威胁+："完美风暴"，可能需要牺牲

## 2. 核心机制详解

### 2.1 拆解与改造系统

**不是制造，而是改造**。

```
传统生存游戏: 收集资源 → 制作新物品
Abiotic Factor: 拆解现有物品 → 改造为工具
```

**拆解示例**：
| 来源物品 | 产出材料 | 设计意图 |
|---------|---------|---------|
| 办公椅 | 木材(2) + 金属管(1) | 日常物品资源化 |
| 电脑主机 | 电路板(1) + 金属碎片(2) | 科技设施可 hack |
| 消防栓 | 金属罐(1) + 压缩气体(1) + 水管(1) | 基础设施武器化 |
| 灭火器 | 金属罐(1) + 灭火剂(1) | 应急设备多功能化 |

**科学武器示例**：
- 改造消防栓 → 高压水枪/电磁陷阱
- 实验室设备 → 分析仪器/防御装置
- 化学试剂 → 腐蚀/冰冻/燃烧手雷

### 2.2 知识解锁机制

**分析流程**：
```
收集样本 → 放入分析机 → 等待研究 → 解锁配方
```

**分析等级**：
- **Level 1**：基础属性（导电性、腐蚀性）
- **Level 2**：行为模式（反应条件）
- **Level 3**：弱点发现（最佳应对）
- **Level 4**：利用方法（制作配方）

**实验性制作**：
```
样本A（属性：腐蚀+辐射）
    +
样本B（属性：电磁+低温）
    =
???（需实验确定，可能有益/有害/灾难性）
```

### 2.3 理智系统：幻觉机制

**理智值设计**：
| 理智状态 | 视觉效果 | 游戏影响 |
|---------|---------|---------|
| 高理智 | 正常视野 | 无影响 |
| 中理智 | 边缘模糊，轻微失真 | 轻微操作延迟 |
| 低理智 | 产生幻觉（看见假的收容物） | 可能误判威胁 |
| 崩溃 | 无法区分真实与幻觉 | 危险行为风险 |

**设计意图**：将心理状态转化为**可玩机制**，而非单纯数值。

### 2.4 涌现性物理交互

**核心公式**：简单规则 + 非生物元素交互 = 无限解法

| 元素A | 元素B | 结果 |
|-------|-------|------|
| 电 | 水 | 电击范围扩大 |
| 电 | 金属 | 传导电击 |
| 火 | 冰 | 蒸汽（视野阻挡）|
| 辐射 | 生物 | 变异/死亡 |
| 重力异常 | 投掷物 | 轨迹改变 |

## 3. 空间与叙事设计

### 3.1 90年代科幻办公美学

**视觉参考**：
- 《半条命》(1998) 的黑山基地
- 米色墙壁、荧光灯、水磨石地板
- CRT显示器、软盘、传真机

**叙事价值**：
- 日常办公用品成为生存资源
- 平凡环境衬托异常恐怖
- 怀旧感增强沉浸

### 3.2 空间层级与风险递进

```
办公区[安全] → 实验室[警戒] → 维护通道[危险] → 深层收容区[高危]
     ↑              ↓              ↓
  通风管道 ← 电梯井 ← 楼梯间
```

| 空间 | 风格 | 功能 | 情感 |
|------|------|------|------|
| **办公区** | 整洁/混乱 | 资源采集 | 熟悉、安全 |
| **实验室** | 高科技 | 研究、制作 | 专业、警戒 |
| **维护通道** | 工业、狭窄 | 捷径、隐藏 | 压抑、危险 |
| **深层收容区** | 异化、未知 | 高风险探索 | 恐怖、神秘 |

### 3.3 环境叙事细节

**办公区线索**（日常悲剧）：
- 倒下的咖啡杯（事发时正在工作）
- 半开的抽屉（有人匆忙找东西）
- 员工照片 + 生日装饰 + 血迹（悲剧发生时正在庆祝）

**深层收容区线索**（异常恐怖）：
- 抓痕墙壁（某物试图逃脱）
- 干涸血迹（多次事故）
- 前员工最后录音
- 邪教符号（疯狂、未知）

## 4. 职业与技能系统

### 4.1 职业专精

| 职业 | 起始技能 | 特殊能力 | 弱点 |
|------|---------|---------|------|
| **理论物理学家** | 物理+5 | 更快解锁高级科技，预测收容物物理行为 | 生物/化学分析慢 |
| **生物学家** | 生物+5 | 更快分析生物样本，与有机收容物建立联系 | 机械/电子维修慢 |
| **工程师** | 机械+5 | 高效拆解/建造，设备维修，临时改造 | 理论分析慢 |
| **防御分析师** | 战术+5 | 威胁评估，安全系统设计，预测收容失效模式 | 进攻性弱 |

### 4.2 技能成长：使用即增强

**设计原理**："你变得擅长某事，是因为你经常做这件事。"

```
频繁使用显微镜 → 观察技能↑
    ↓
更快分析样本
    ↓
解锁高级观察选项
    ↓
发现隐藏细节
```

## 5. 多人协作设计

### 5.1 角色分工

**管理员/指挥**：全局监控、资源分配、决策制定
**外勤/探索**：外出采集、危险区域侦察、紧急救援
**研究员/分析**：样本分析、科技研发、信息解读
**维护/工程**：设施维护、电力系统、防御建设

### 5.2 技能互补

**示例场景**：
```
发现新收容物
    ↓
外勤：采集样本（高风险）
    ↓
研究员：分析样本（需要时间）
    ↓
工程师：制作对应装备（需要配方）
    ↓
全员：使用新装备镇压收容物
```

**1+1+1+1 > 4**：涌现性协作效果。

## 6. 对2D俯视角游戏的移植启示

### 6.1 核心保留

1. **拆解系统**：鼠标悬停显示可拆解，右键拆解
2. **知识解锁**：先分析才能制作对应装备
3. **环境利用**：利用场景元素而非纯战斗
4. **理智系统**：视觉扭曲而非单纯数值
5. **职业差异**：不同起始技能，使用成长

### 6.2 2D特有优化

1. **视野受限**：Fog of War增强恐怖感
2. **像素艺术**：强化复古SCP文档感
3. **简化物理**：2D碰撞/触发式交互替代3D模拟
4. **快速迭代**：小团队更容易修改

### 6.3 简化建议

**保留核心**：
- 拆解/改造
- 知识解锁
- 理智/压力
- 环境叙事

**可简化**：
- 复杂物理 → 触发式交互
- 精细建模 → 像素艺术
- 语音对话 → 文本日志
- 大型设施 → 模块化房间

## 7. 设计检查清单

### 身份重构
- [ ] 玩家是否感到"弱小但聪明"？
- [ ] 是否有"用科学解决问题"的时刻？
- [ ] 环境资源是否比野外资源更重要？

### 非生物威胁
- [ ] 环境威胁是否创造紧张感？
- [ ] 威胁叠加是否有策略深度？
- [ ] 玩家是否有多种应对方式？

### 叙事设计
- [ ] 场景是否讲述"这里发生过什么"？
- [ ] 平凡环境是否衬托异常恐怖？
- [ ] 玩家能否拼凑出完整故事？

### 知识系统
- [ ] 分析-解锁-制作的循环是否 satisfying？
- [ ] 知识是否带来真正的力量感？
- [ ] 未知是否既危险又有吸引力？

## 9. 对比案例

### vs 脑叶公司 (CASE-002)

| 维度 | Abiotic Factor | 脑叶公司 |
|------|---------------|---------|
| **视角** | 第一人称生存 | 管理者俯视 |
| **恐怖来源** | **环境压迫**（饥饿/氧气/怪物） | **认知不确定**（不知道规则） |
| **核心循环** | 探索→收集→建造→生存 | 管理→收容→研究→应对危机 |
| **资源逻辑** | 物理资源（材料、食物） | 抽象资源（PE-Box、员工理智） |
| **压力机制** | 生理指标持续下降 | Qliphoth倒计时、知识锁 |

**关键差异**：两者都使用"死亡→学习→再尝试"的循环，但脑叶将"知识"本身作为核心资源，而AF将"物资"作为核心资源。
