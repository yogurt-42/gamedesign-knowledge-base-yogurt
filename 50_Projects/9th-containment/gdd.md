# 第9号收容所 - 游戏设计文档

## 项目概述

**类型**: 2D俯视角生存 + 收容管理 + 多人合作  
**风格**: 饥荒 × 脑叶公司 × 非生物因素  
**平台**: PC (Steam/itch.io)

## 核心循环

```
白天阶段:
├── 探索设施 → 收集资源
├── 维护收容单元 → 管理SCP
└── 研发升级 → 解锁新能力

夜晚阶段 (高危):
├── 收容压力增加
├── 可能突破收容
└── 紧急事件响应
```

## 独特卖点 (USP)

1. **科学家身份**: 用知识而非武力生存
2. **涌现性系统**: 元素交互产生无限解法
3. **环境叙事**: 拼凑前员工的故事
4. **多人分工**: 管理员/外勤/研究员角色
