# Unfinished - 时间循环太空冒险游戏

## 项目概述

一款以**时间循环**为核心机制的太空冒险文字游戏。玩家被困在即将面临"审判日"的星系中，通过反复循环积累情报、建立关系、收集关键物品，最终面对终局挑战。

## 核心设计原则

1. **小而真实的星球** - 不做 procedurally generated 的空洞世界，每颗星球都是手工打造的微型社会
2. **AI 友好型开发** - 所有内容使用 Markdown 编写，便于 AI 理解和扩展
3. **文字优先** - 技术力有限，但文字比图像更有感染力
4. **设定优先** - 先构建可信的世界，再谈游戏机制

## 核心机制

### 时间循环
- **循环触发**：审判日到来即重置（被动）
- **保留内容**：情报、Key 物品、飞船日志
- **重置内容**：物资、普通装备、关系进度

### 关系系统
- 关系每次循环重置，但**情报让你"作弊"**
- 知道 NPC 的秘密 → 快速建立信任
- 知道最佳时机 → 在关键时刻出现
- 利用信息不对称创造叙事张力

### 能力锁
- 不是纯知识锁（Outer Wilds 式）
- 使用 **Key 物品**作为能力锁钥匙：
  - 古老星图 → 导航到隐藏星球
  - 髓晶共鸣器 → 激活古代装置
  - 信任信物 → 进入势力禁区

## 世界观基础

### 核心资源：髓晶
- 魔幻矿物，既是能源也是权力
- 受管制、昂贵、危险
- 开环设计 = 性能换污染，道德困境

### 引擎厂商 = 文化势力
| 势力 | 气质 | 设计局文化 |
|------|------|-----------|
| 星迹集团 | 商业化、国有化、务实 | 中式企业官僚 |
| 科产联"未来" | 苏联式理想主义、衰落中 | 俄式航天浪漫 |
| 马克探索技术 | 快速迭代、冒险、不稳定 | 美式硅谷风格 |
| 黎明设计局 | 稳健、长程规划 | 欧陆工程师 |
| 星际客车公司 | 可靠、服务、中庸 | 日式服务业 |
