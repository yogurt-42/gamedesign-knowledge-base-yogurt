# 游戏设计知识库 | Game Design Knowledge Base

> 从零开始设计游戏的系统性资料整理，聚焦生存、收容、涌现性玩法

Systematic knowledge base for game design, focusing on survival, containment, and emergent gameplay.

---

## 📦 内容概览 | Content Overview

这份知识库整理了从学术理论到实践模板的完整设计资料，特别聚焦于：

This knowledge base compiles complete design resources from academic theory to practical templates, with special focus on:

- **生存游戏设计** - 核心循环、经济系统、心流理论
  *Survival Game Design* - Core loops, economic systems, flow theory
  
- **收容物/异常设计** - SCP风格框架、临床语调、分级系统
  *Containment/Anomaly Design* - SCP-style framework, clinical tone, classification
  
- **涌现性玩法** - 系统式设计、元素交互、非预期解法
  *Emergent Gameplay* - Systemic design, element interaction, unexpected solutions
  
- **环境叙事** - 指示符叙事、场景设计、碎片化故事
  *Environmental Storytelling* - Indexical narrative, scene design, fragmented stories

## 🚀 快速开始 | Quick Start

### 对于策划 | For Designers
1. 从 `theory/survival-game-blueprint.md` 理解核心循环
2. 参考 `core-mechanics/abiotic-factor-*.md` 确定独特卖点
3. 使用 `references/gdd-template.md` 撰写设计文档

### 对于开发者 | For Developers
1. 查看 `references/2d-topdown-guide.md` 技术架构
2. 参考 `systems/systemic-design.md` 实现涌现性
3. 使用 `core-mechanics/scp-framework.md` 设计实体

### 对于叙事 | For Writers
1. 学习 `narrative/environmental-storytelling.md` 场景设计
2. 应用 `theory/scp-official-standards.md` 文档语调
3. 参考 `core-mechanics/abiotic-factor-deep-dive.md` 空间叙事
