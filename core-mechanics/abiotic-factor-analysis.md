# 非生物因素 (Abiotic Factor) 深度分析

## 核心身份重构

### 科学家 vs 战士
| 传统生存游戏 | 非生物因素 | 设计启示 |
|-------------|-----------|---------|
| 捡石头做石斧 | 拆解办公椅获金属管 | 资源来自设施拆解 |
| 打猎获生肉 | 离心机分离营养膏 | 科技加工而非原始采集 |
| 造盔甲防御 | 改造消防栓做电磁炮 | 利用环境而非制造 |

**关键洞察**：玩家是科研人员，核心能力是知识应用而非武力。

## 三维生存压力

### 生理指标设计
| 指标 | 来源 | 效果 |
|------|------|------|
| 饥饿 | 营养膏/合成食品 | 基础生存 |
| 口渴 | 净水系统 | 设施依赖 |
| 健康 | 辐射/疾病 | 长期伤害 |
| 排泄 | 真实到搞笑 | 沉浸感 |
| **理智** | 看见异常物 | **幻觉系统** |

**理智值设计**：
- 高理智：正常视野
- 中理智：边缘模糊，轻微失真
- 低理智：产生幻觉（看见假的收容物）
- 崩溃：无法区分真实与幻觉

## 地图设计：3D箱庭的2D转化

### 空间层级
```
办公区[安全] → 实验室[警戒] → 维护通道[危险] → 深层收容区[高危]
     ↑              ↓              ↓
  通风管道 ← 电梯井 ← 楼梯间
```

### 关键原则
- 非线性探索
- 垂直通道连接
- 风险与奖励正比

## 应用到2D俯视角

### 核心移植点
1. **拆解光标**：鼠标悬停显示可拆解物体
2. **知识解锁**：先分析才能制作对应装备
3. **环境交互**：利用场景元素而非纯战斗
4. **理智系统**：视觉扭曲而非单纯数值

### 2D特有优势
- 视野受限（Fog of War）增强恐怖感
- 更容易实现"指示符叙事"
- 像素艺术可强化复古SCP文档感
