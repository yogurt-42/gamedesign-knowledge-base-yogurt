# 非生物因素专项设计

## 核心设计理念

**《非生物因素》(Abiotic Factor)**  
开发商：Deep Field Games  
发售时间：2024年5月（EA版）

**核心突破**：生存游戏的身份重构

---

## 科学家而非战士

### 身份反转设计

| 传统生存游戏 | 非生物因素 | 设计意义 |
|-------------|-----------|---------|
| 战士/探险家 | 科学家/研究员 | 知识即武器 |
| 蛮力征服 | 科学利用 | 动脑而非动手 |
| 野外求生 | 设施内生存 | 工业资源而非自然资源 |

### 物理利用机制

**环境拆解**：
```
办公桌 → 木材(2) + 金属管(1)
电脑主机 → 电路板(1) + 金属碎片(2)
消防栓 → 金属罐(1) + 压缩气体(1) + 水管(1)
灭火器 → 金属罐(1) + 灭火剂(1)
```

**改造而非制造**：
- 传统：收集资源 → 制作新物品
- 非生物因素：拆解现有物品 → 改造为工具

**科学武器示例**：
- 改造消防栓 → 高压水枪
- 实验室设备 → 电磁陷阱
- 化学试剂 → 腐蚀/冰冻手雷

## 非生物威胁设计

### 环境危害类型

| 威胁 | 机制 | 应对策略 |
|------|------|---------|
| **寒流** | 温度骤降，快速失温 | 供暖设备、保温服 |
| **供电中断** | 收容单元失效 | 备用发电机、手动维护 |
| **黑雾** | 视野受限，理智下降 | 光源、快速撤离 |
| **辐射** | 持续伤害，变异风险 | 铅屏蔽、抗辐射药 |
| **不稳定重力** | 跳跃/移动受影响 | 重力锚定设备 |
| **气压变化** | 门无法开启/突然关闭 | 气压调节、备用通道 |

### 威胁叠加设计

**单一威胁**：可管理  
**双重威胁**：需要优先级判断  
**三重威胁+**：紧急状态，可能需要牺牲

**示例**：
- 寒流 + 供电中断 = 供暖失效，快速失温
- 辐射 + 黑雾 = 无法看到辐射源，双重恐惧
- 所有威胁同时 = "完美风暴"，团队可能团灭

## 办公空间叙事

### 90年代科幻美学

**视觉参考**：
- 《半条命》(1998) 的黑山基地
- 米色墙壁、荧光灯、水磨石地板
- CRT显示器、软盘、传真机

**叙事价值**：
- 日常办公用品成为生存资源
- 平凡环境衬托异常恐怖
- 怀旧感增强沉浸

### 空间对比设计

| 空间 | 风格 | 功能 | 情感 |
|------|------|------|------|
| **办公区** | 整洁/混乱 | 资源采集 | 熟悉、安全 |
| **实验室** | 高科技 | 研究、制作 | 专业、警戒 |
| **维护通道** | 工业、狭窄 | 捷径、隐藏 | 压抑、危险 |
| **深层收容区** | 异化、未知 | 高风险探索 | 恐怖、神秘 |

### 环境叙事细节

**办公区线索**：
- 倒下的咖啡杯（事发时正在工作）
- 半开的抽屉（有人匆忙找东西）
- 员工照片（人性化受害者）
- 生日装饰 + 血迹（悲剧发生时正在庆祝）

**深层收容区线索**：
- 抓痕墙壁（某物试图逃脱）
- 干涸血迹（多次事故）
- 前员工最后录音
- 邪教符号（疯狂、未知）

## 科学制作系统

### 知识解锁机制

**分析流程**：
```
收集样本 → 放入分析机 → 等待研究 → 解锁配方
```

**分析等级**：
- Level 1：基础属性（导电性、腐蚀性）
- Level 2：行为模式（反应条件）
- Level 3：弱点发现（最佳应对）
- Level 4：利用方法（制作配方）

### 实验性制作

**不确定结果**：
```
样本A（属性：腐蚀+辐射）
    +
样本B（属性：电磁+低温）
    =
???（需实验确定）
```

**可能结果**：
- 能量释放（爆炸/能量源）
- 中和反应（无害化）
- 新物质（未知属性，需进一步分析）
- 灾难（实验事故）

**实验风险**：
- 低等级样本：相对安全
- 高等级样本：需要防护措施
- 危险样本：远程操作、隔离室

## 身份系统：职业差异

### 职业类型（示例）

| 职业 | 起始技能 | 特殊能力 |
|------|---------|---------|
| **理论物理学家** | 理论+5 | 更快解锁高级科技 |
| **防御分析师** | 战术+5 | 更好识别威胁模式 |
| **生物学家** | 生物+5 | 更快分析生物样本 |
| **工程师** | 机械+5 | 更高效拆解/建造 |

### 技能成长：使用即增强

**设计**：
- 频繁使用某类工具 → 该技能提升
- 制作某类物品 → 相关技能提升
- 分析某类样本 → 专业技能提升

**示例**：
```
玩家经常使用电击武器
    ↓
电学技能提升
    ↓
解锁：高级电击陷阱、EMP手雷
```

## 多人协作：科学家团队

### 角色分工

**管理员/指挥**：
- 全局监控
- 资源分配
- 决策制定

**外勤/探索**：
- 外出采集
- 危险区域侦察
- 紧急救援

**研究员/分析**：
- 样本分析
- 科技研发
- 信息解读

**维护/工程**：
- 设施维护
- 电力系统
- 防御建设

### 技能互补

**示例场景**：
```
发现新收容物
    ↓
外勤：采集样本（高风险）
    ↓
研究员：分析样本（需要时间）
    ↓
工程师：制作对应装备（需要配方）
    ↓
全员：使用新装备镇压收容物
```

## 给你的2D游戏：移植要点

### 核心保留

1. **拆解系统**：鼠标悬停显示可拆解，右键拆解
2. **知识解锁**：先分析才能制作
3. **环境利用**：利用场景而非纯战斗
4. **理智系统**：视觉扭曲而非单纯数值
5. **职业差异**：不同起始技能，使用成长

### 2D特有优化

1. **视野受限**：Fog of War增强恐怖感
2. **像素艺术**：强化复古SCP文档感
3. **简化物理**：2D碰撞而非3D模拟
4. **快速迭代**：小团队更容易修改

### 简化建议

**保留核心**：
- 拆解/改造
- 知识解锁
- 理智/压力
- 环境叙事

**可简化**：
- 复杂物理（改为触发式）
- 精细建模（改为像素）
- 语音对话（改为文本）
- 大型设施（改为模块化房间）

## 关键设计检查清单

### 身份重构
- [ ] 玩家是否感到"弱小但聪明"？
- [ ] 是否有"用科学解决问题"的时刻？
- [ ] 环境资源是否比野外资源更重要？

### 非生物威胁
- [ ] 环境威胁是否创造紧张感？
- [ ] 威胁叠加是否有策略深度？
- [ ] 玩家是否有多种应对方式？

### 叙事设计
- [ ] 场景是否讲述"这里发生过什么"？
- [ ] 平凡环境是否衬托异常恐怖？
- [ ] 玩家能否拼凑出完整故事？
